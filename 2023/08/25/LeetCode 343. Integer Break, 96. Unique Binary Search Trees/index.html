343. Integer Break

Link: https://leetcode.com/problems/integer-break/

Description

Given an integer n, break it into the sum of k positive integers, where k >= 2, and maximize the product of those integers.

Return the maximum product you can get.

Approach

  • Initialize a dp array, which is used to store the maximum product of each number.
  • Set dp[2] equals to 1.
  • For i in range [3,n]:
    • For j in range [1, i / 2]:
      • The product can be get by j * (i - j) and j * dp[i - j], so we need to compare the two products and the original value of dp[i].
  • Return dp[n].
